## はじめに

TSUKASA AIは、複数のメッセージングチャネル（LINE、Slack、Discord、Telegram など）を統合し、AIアシスタントとの対話を一元管理するゲートウェイです。

### 対応チャネル

| チャネル    | 説明                               |
| ----------- | ---------------------------------- |
| LINE        | LINE公式アカウント経由             |
| Slack       | Socketモードによるリアルタイム連携 |
| Discord     | Botによるサーバー連携              |
| Telegram    | Botによるチャット連携              |
| Google Chat | Webhook APIによる連携              |
| Signal      | signal-cliによるメッセージ連携     |
| WhatsApp    | WhatsApp Webブリッジ               |
| iMessage    | macOSブリッジ                      |
| Nostr       | 分散型DM（NIP-04）                 |

## ログイン

### パスワード認証の場合

1. ブラウザで管理画面URL（例: `https://your-domain.example.com`）にアクセス
2. 管理者から共有されたパスワードを入力
3. 「接続」をクリック

### SSO認証の場合

1. ブラウザで管理画面URLにアクセス
2. 「SSOでログイン」をクリック
3. 組織のIDプロバイダー（Okta、Azure AD等）で認証
4. 自動的に管理画面にリダイレクト

## チャット

### 基本的な使い方

1. 左側ナビゲーションから「Chat」を選択
2. メッセージ入力欄にテキストを入力
3. Enter キーまたは送信ボタンでメッセージを送信
4. AIからの応答が表示されます

### セッション

- 各会話は「セッション」として管理されます
- 新しいセッションを開始するには、セッションキーを変更します
- 過去のセッションは「Sessions」タブから確認できます

### 添付ファイル

メッセージに画像やファイルを添付して送信できます。

## チャネル管理

### チャネルの状態確認

1. 左側ナビゲーションから「Channels」を選択
2. 各チャネルのカードに状態が表示されます：
   - **設定済み** — チャネルの設定が完了している
   - **稼働中** — チャネルが動作している
   - **接続済み** — 外部サービスとの接続が確立している

### チャネル設定の変更

1. 対象チャネルのカードを開く
2. 設定フォームで値を変更
3. 「保存」ボタンをクリック

### WhatsApp の連携

1. Channels画面でWhatsAppカードを開く
2. 「QR表示」ボタンをクリック
3. スマートフォンのWhatsAppアプリでQRコードをスキャン
4. 連携が完了すると「連携済み」と表示されます

## セッション管理

### セッション一覧の確認

1. 「Sessions」タブを選択
2. フィルター機能でセッションを絞り込み：
   - アクティブなセッションのみ
   - 特定の期間
   - Cronセッションの除外

### セッションの操作

- **詳細表示** — セッションをクリックして会話履歴を確認
- **削除** — 不要なセッションを削除

## 設定

### 言語設定

1. 画面右下の言語メニューを開く
2. 日本語（Japanese）を選択
3. 画面が日本語に切り替わります

### テーマ設定

画面右上のテーマ切り替えボタンで変更できます：

- **システム** — OSの設定に合わせる
- **ライト** — 明るいテーマ
- **ダーク** — 暗いテーマ

### ナビゲーション

- 左上のメニューボタンでナビゲーションバーの折りたたみ/展開ができます
- ナビゲーションのグループは折りたたみ可能です

## よくある操作

### AIモデルの切り替え

管理者が設定した複数のAIモデルから選択できます：

1. 「Config」タブでエージェント設定を開く
2. モデルを変更して「保存」

### スキルの管理

1. 「Skills」タブを選択
2. 利用可能なスキルの一覧が表示されます
3. スキルのインストール/有効化/無効化ができます

### Cronジョブ

定期実行タスクの確認と管理：

1. 「Cron」タブを選択
2. ジョブの一覧と実行履歴を確認
3. ジョブの有効化/無効化、手動実行ができます
