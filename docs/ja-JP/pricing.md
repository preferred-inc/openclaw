# 料金体系・プラン設計

> **注意**: この料金体系はテンプレートです。実際の料金は運営企業が決定してください。

---

## 概要

TSUKASA AI（OpenClaw）はセルフホスト型のAIゲートウェイです。ソフトウェア自体はMITライセンスで無償提供されますが、運用にあたっては以下のコストが発生します。

## コスト構成

### 1. インフラコスト（必須）

| 項目      | 目安               | 備考                              |
| --------- | ------------------ | --------------------------------- |
| サーバー  | ¥2,000〜¥20,000/月 | VPS/クラウド（AWS, GCP, Azure等） |
| ドメイン  | ¥1,000〜¥5,000/年  | 独自ドメイン                      |
| SSL証明書 | 無料〜             | Let's Encrypt推奨                 |

### 2. AIプロバイダーコスト（従量課金）

| プロバイダー | モデル            | 入力            | 出力             |
| ------------ | ----------------- | --------------- | ---------------- |
| OpenAI       | GPT-4o            | $2.50/1M tokens | $10.00/1M tokens |
| OpenAI       | GPT-4o-mini       | $0.15/1M tokens | $0.60/1M tokens  |
| Anthropic    | Claude 3.5 Sonnet | $3.00/1M tokens | $15.00/1M tokens |
| Anthropic    | Claude 3.5 Haiku  | $0.80/1M tokens | $4.00/1M tokens  |

> ※ 料金はプロバイダーの最新の料金表をご確認ください。

### 3. メッセージングチャネルコスト

| チャネル | コスト           | 備考                             |
| -------- | ---------------- | -------------------------------- |
| LINE     | 無料プラン〜     | メッセージ数による従量課金あり   |
| Slack    | Slackプラン依存  | ワークスペースのプランに含まれる |
| Discord  | 無料             | Bot利用は無料                    |
| Telegram | 無料             | Bot APIは無料                    |
| WhatsApp | Business API費用 | メッセージ単価あり               |

## プラン例

### フリープラン（セルフホスト）

- **料金**: 無料（インフラ・AI API費用は利用者負担）
- **内容**:
  - ソフトウェアの無制限利用
  - 全チャネル対応
  - コミュニティサポート（GitHub Issues）
- **制限**:
  - サポートなし
  - SLAなし

### スタンダードプラン

- **料金**: ¥50,000/月〜（目安）
- **内容**:
  - フリープランの全機能
  - メールサポート（営業日対応）
  - セットアップ支援
  - 月次レポート
- **対象**: 小〜中規模チーム（10人以下）

### エンタープライズプラン

- **料金**: 個別見積もり
- **内容**:
  - スタンダードプランの全機能
  - SSO/SAML認証
  - RBAC（権限管理）
  - 監査ログ
  - IP制限
  - 専任サポート（SLA付き）
  - カスタマイズ対応
  - オンボーディング支援
- **対象**: 大規模組織（10人以上）

## 課金モデルの選択肢

### A. ユーザー数ベース

| ユーザー数 | 月額（税抜） |
| ---------- | ------------ |
| 〜5人      | ¥30,000      |
| 〜20人     | ¥80,000      |
| 〜50人     | ¥150,000     |
| 51人〜     | 個別見積もり |

### B. 従量課金ベース

| 項目           | 単価                |
| -------------- | ------------------- |
| AIリクエスト   | ¥1〜¥10/リクエスト  |
| メッセージ配信 | ¥0.5〜¥5/メッセージ |
| ストレージ     | ¥100/GB/月          |

| ------------ | ------------ | -------------------- |
